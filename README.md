# 🍽️ CFFMS Vendor Dashboard

Vendor-side module for the **CFFMS (Cafeteria Food & Facility Management System)** that enables cafeteria vendors to manage food orders, track token queues, and control menu availability through a modern dashboard interface.

---

## 📌 Overview

The Vendor Dashboard helps cafeteria vendors efficiently manage daily operations by providing tools to monitor incoming orders, update preparation status, and manage menu items.

The interface is designed with a **modern responsive UI**, smooth animations, and an intuitive workflow to simplify order management.

---

## 🚀 Features

* 🔐 Vendor Login Authentication
* 📊 Order Queue & Token Tracking
* 🍔 Menu Item Management
* 📈 Basic Order Statistics
* 🎨 Modern Dashboard UI (Dark Mode)
* 📱 Responsive Design

---

## 🛠️ Tech Stack
